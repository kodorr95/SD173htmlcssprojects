Borders

The border property allows you to specify the style, width, and color of an element's border

Border style

The border-style property can take up to four values: top, right, bottom, left (all borders). Here are some examples of what is allowed.

Border Width

NOTE: border-width will not work unless you have a border style set

Border-width can be set to a specific size (px, em, etc) or you can use one of the following pre-sets: thin, medium, thick. Border-width property can take up to four directional values: T, R, B, L .

Border Color

NOTE: border-color will not work unless the border-style is set.

Border color can be set in the following ways:

  • Color name
  • Hex: #FF8855
  • RGB
  • transparent
