CSS IDs

An ID is a UNIQUE identifier

ID Notes

  • In CSS an ID starts with a octothorpe (crunch)
  • Example uses for IDs: Targeting a specific header or about section
  • Each ID should only be used ONCE per page - they are UNIQUE
    This is important with Javascript later on
  • Use for one-off styling

IDs versus Classes

IDs

This id will be linked in the CSS page by an #

You cannot use two ids on one element. If you do, neither ID will be applied

Classes

Elements can have any number of classes

For example, this element has two classes: "first-class" and "second-class"

You CAN TECHNICALLY use an ID twice - VS code will not stop you and you won't get any errors, but YOU SHOULD NOT.
